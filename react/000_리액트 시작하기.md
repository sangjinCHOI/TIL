# 000_리액트 시작하기

### 1. 개발환경 구축

1. node.js 설치
   [**https://nodejs.org/ko/**](https://nodejs.org/ko/) 에서LTS버전으로 다운로드
2. cmd창에 `npm i -g create-react-app` 으로 create-react-app 설치



### 2. app 설치하기

1. 설치할 폴더 위치에서 cmd창 열기

2. `create-react-app <project name>` 으로  시작하기

3. 정상적으로 프로젝트가 설치 되었는지 확인하기
   ```bash
   cd <project name>
   npm start
   ```
